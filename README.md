# Project Setup and Run Instructions

## How to Run

1. Navigate to the directories:
   - `cd app1/`
   - `cd app2/`

2. In **app1**, start the remote server:
   ```bash
   npm run prod
   
3. In **app 2**, start the development server:
   ```bash
   npm run dev
